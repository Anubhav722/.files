# .files
My brand new .files (dotfiles).

## Setup

```
cd ~
git clone git@github.com:dhilipsiva/.files.git
pip3 install tmuxp powerline-shell
cp .files/home/.* ~
Install NerdFonts: https://github.com/ryanoasis/nerd-fonts
Install Powerline Fonts: https://github.com/powerline/fonts
Configure your vim and terminal fonts to any font that has "powerline" as a part of its name
Install brew / linuxbrew
brew install neovim bash
Install Vim Plug for NeoVIM: https://github.com/junegunn/vim-plug
```
